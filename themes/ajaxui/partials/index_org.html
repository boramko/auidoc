jindo js는 nhn에서 제작한 javascript library입니다.

indoJS Component는 JindoJS 를 기반으로 개발된 재사용 가능한 독립적인 자바스크립트 모듈입니다. JindoJS는 Javascript Application 제작시에 반복적으로 발생하는 로직을 컴포넌트화 하여 결과물의 품질을 향상시키기 위한 목적으로 작성되었습니다.
